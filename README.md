# 🥷⚡ JobNinja SuperSaiyan

AI-inspired job search system that runs 100% in the browser — no server needed.

## Features

- **🔍 Job Search** — Live remote job listings via Remotive API (no key needed)
- **⚡ Job Scorer** — Rate jobs across 10 dimensions, get an A–F score + verdict
- **📋 Application Tracker** — Kanban board (Applied → Interview → Offer → Rejected), saved in your browser
- **📊 Stats Dashboard** — Interview rate, offer count, grade distribution
- **⬇ CSV Export** — Export all tracked applications

## Deploy to GitHub Pages

1. Push this repo to GitHub
2. Go to Settings → Pages → Source: Deploy from branch → `main` → `/root`
3. Live at `https://YOUR_USERNAME.github.io/job-ninja-supersaiyan`

## Data

All data is stored in `localStorage` — private to your browser, never sent anywhere.
